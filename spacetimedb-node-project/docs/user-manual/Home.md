## 🚀 Quick Concept Check

*   **Venue**: Your physical space or team (e.g., "Main Lobby", "Sales Dept").
*   **Channel**: A specific topic for messages (e.g., "Alerts", "Promotions").
*   **Display Node**: A computer/TV running the Canal4 desktop app that shows your messages.
*   **Broadcast**: The actual message sent to the scrolling tickers.
