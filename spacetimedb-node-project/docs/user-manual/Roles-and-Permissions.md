# Roles and Permissions

Canal4 has a granular permission system to control who can manage the venue and send broadcasts.

## 👥 Understanding Roles

There are four primary roles in a Venue:

| Role | Permissions |
|---|---|
| **Owner** | Full control. Can delete the venue, manage all settings, and assign any role to anyone. |
| **Admin** | Can manage channels, display nodes, and invite new members. Can assign 'Moderator' or 'Member' roles. |
| **Moderator** | Can send broadcasts and repeat past messages. Can delete messages from history. |
| **Member** | View-only access by default. Can see the message history but cannot send broadcasts. |

## 📩 Inviting Members

To grow your team:
1.  Open the menu (⋮) and select **"Invite"**.
2.  **Share the Link**: Copy the invitation URL or show the **QR Code**.
3.  New users who click the link will automatically join the venue as **Members**.

## 🛡️ Managing Permissions

Owners and Admins can promote or demote users:
1.  Select **"Permissions"** from the venue menu.
2.  Search for the member.
3.  **Update Role**: Change their base role in the venue.
4.  **Channel Overrides**: You can give someone "Moderator" access to a specific channel while leaving them as a "Member" elsewhere.

## 🚫 Blocking Users
If a user should no longer have access to the venue:
1.  Go to the member's profile in the **Permissions** screen.
2.  Click **"Block User"**. This immediately revokes their access to all channels and messages in this venue.
3.  You can **Unblock** them later if needed.
