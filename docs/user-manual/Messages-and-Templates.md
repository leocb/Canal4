# Messages and Templates

Canal4 uses **Templates** to ensure broadcast messages are consistent, well-formatted, and easy to input.

<a name="templates"></a>
## 📋 Using Templates

A template defines the structure of a message. For example, a "Price Update" template might have fields for "Product" and "New Price".

### Why use Templates?
*   **Consistency**: Every message for a specific purpose looks the same.
*   **Speed**: Staff just fill in the blanks instead of typing full sentences.
*   **Validation**: Ensure staff enter numbers where numbers are expected.

### Creating a Template
1.  Inside a Channel, click **"Templates"**.
2.  Click **"Create Template"**.
3.  **Add Fields**: Each field can have:
    *   **Label**: What the user sees (e.g., "Price").
    *   **Prefix/Suffix**: Text that appears before/after the value (e.g., "$" or " per kg").
    *   **Rules**: Make a field numeric-only or optional.

<a name="broadcasting"></a>
## 📢 Sending Broadcasts

Once a template is created, you can send a broadcast.

1.  Navigate to a Channel.
2.  Click **"Send New Broadcast"**.
3.  **Select a Template**: Choose the structure you need.
4.  **Fill the Fields**: The **Live Preview** at the bottom shows exactly how it will scroll on the display nodes.
5.  **Send**: Click send. The message will instantly "pop" onto the scrolling tickers of all displays paired to this venue.

## 🔁 Repeating Messages
If you want to send a previous message again:
1.  Find the message in the channel history.
2.  Click **"Display Again"**. This re-broadcasts the message to all active displays.

## 🗑️ Managing History
Moderators and Admins can delete messages from the history if they were sent in error.
