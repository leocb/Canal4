# Display Nodes and Pairing

Display Nodes are the heart of Canal4. They are the scrolling tickers that show your messages on physical screens.

## 📺 Setting up a Display Node

To set up a new screen:
1.  **Download & Install**: Install the Canal4 desktop application on the computer connected to your TV/Monitor.
2.  **Launch**: Open the app. It will show a **6-digit Pairing PIN** on the screen.
3.  **Connectivity Check**: Ensure the computer has internet access and can reach your SpacetimeDB server.

## 🔗 The Pairing Process

Pairing links the physical screen to your specific Venue.

1.  In the **Web Dashboard**, navigate to the Venue where you want to add the screen.
2.  Open the menu (⋮) and select **"Display Nodes"**.
3.  Click **"Add Node"**.
4.  **Enter the PIN**: Type in the 6-digit code shown on the desktop screen.
5.  **Name the Node**: Give it a name (e.g., "Lobby Main TV") so you can identify it later.
6.  **Confirm**: Click Pair. Within seconds, the desktop app will sync and start showing the venue's ticker.

## 🛠️ Managing Displays

From the **Display Nodes** screen in the web dashboard, you can:
*   **Monitor Status**: See if a screen is currently `Connected` or `Offline`.
*   **Rename**: Change the identified name of the display.
*   **Unpair/Delete**: Remove a display from your venue. The desktop app will return to its pairing screen.

## 💡 Best Practices
*   **Wired Connection**: For mission-critical tickers, use a wired Ethernet connection to prevent flickering/delays.
*   **Kiosk Mode**: Set the desktop computer to "Auto-Login" and "Auto-Launch" the Canal4 app on startup.
